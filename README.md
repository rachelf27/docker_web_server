# Web Server with NodeJs using Docker
This is a simple Node.js application using Express framework and Bootstrap.
This application is a simple website for Low Carb Diet Lifestyle.

## Directory Structure 
``` 
 |--node_modules  
 |--public  
    |--css 
       |--styles.css 
    |--asserts
       |--home_page.html
       |--low_carb_diet.html
 |--views
 |--Dockerfile
 |--.dockerignore
 |--package.json
 |--package-lock.json
 |--server,js
 |--README.md
```
